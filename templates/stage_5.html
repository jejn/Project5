Stage 5: The Next Step

Front End Web development

In Intro to HTML and CSS, I learned:

  • To structure a page with a grid layout
  • Working with a mock-up
  • An introduction to Bootstrap and frameworks in general

Additional resources I used for this was the Bootstrap documentation.

Responsive Web Design

In Responsive Web Design Fundamentals, I learned:

  • Why it's important to think about how content look on different devices and screens
  • What different tools are available to test how content look on different devices
  • About Hardware pixels, Device independent pixels, viewport and pixel ratios
  • How to make pages scale better using relative widths
  • How to work with Media queries, Flex box and Breakpoints to make responsive pages

Additionally, I've changed the design on this website to be more responsive. The Breakpoints are at 700px and 960px. The recourses I used was mainly: w3cschools.com and stackoverflow, as well as the lessons from General Assembly on responsive design.

JavaScript

In JavaScript Basics, I learned:

  • To work with the console.log() to debug and check code
  • How to create variables, arrays and objects
  • The basics of JSON (JavaScript Object Notation)
  • To use general expressions to work with strings
  • The difference with dot notation and bracket notation
  • To work with if, while and for statements in JavaScript
  • To create functions and working with jQuery
  • What Encapsulation means

Recourses I used to solve the quizzes and further my understanding were: json.org and MDN. Additionally, I added a JavaScript navigation bar for the small screens layout.

Intro to jQuery

In Intro to jQuery, I learned:

  • That jQuery is a JavaScript library
  • How to navigate the DOM with traversal methods
  • How to manipulate the DOM with jQuery
  • To use the jQuery documentation for reference

To solve the quizzes I used the jQuery Documentation.

Intro to AJAX

In Intro to AJAX, I learned:

  • What AJAX is and what it can be used for
  • To send AJAX requests
  • To incorporate APIs on a website
  • To parse JSON responses and to use the data received

For this course I also used the jQuery Documentation.

Web Development

In general, a lot of things were brush-ups on what I learned in Stage 4, most of which would actually have been very useful for that stage. There were a lot of things that became clear now that I was struggling with then. The additional things I learned in this course were:

  • What Cookies are and how to use them
  • How Hashing is usefully and how it can be applied to increase security
  • "Hashing" and "Salting" passwords
  • Bcrypt and HTTPS
  • Caching, Memcached and a better understanding of how servers, apps and databases interact

Additionally, I've included caching in my own Web App.

Intro to Relational Databases

In Intro to Relational Databases, I learned:

  • How to use SELECT statements to get data from a DB
  • How to do Joins to use multiple tables to gather data
  • How to update and delete entries in a DB
  • What Normalized table design is
  • The concept of declared relationships using primary and foreign keys
